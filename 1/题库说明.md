# 题库说明文档

## 题库结构

本题库共分为6个分类：
1. 建筑题库 - 50题（30单选 + 20多选）
2. 生存题库 - 50题（25单选 + 25多选）
3. 养老题库 - 60题（30单选 + 30多选）
4. 红石题库 - 60题（30单选 + 30多选）
5. 指令题库 - 60题（30单选 + 30多选）
6. 服务器规则题库 - 30题（20单选 + 10多选）

## 必答题规则

### 重要说明
- 每个题库的单选题和多选题部分都包含至少1道必答题
- 必答题在表格中标记为"是"
- 答错任意一道必答题将直接淘汰，需要重新答题
- 必答题是测试的核心内容，请认真作答

### 必答题分布
- 建筑题库：单选必答1题，多选必答1题
- 生存题库：单选必答1题，多选必答1题
- 养老题库：单选必答1题，多选必答1题
- 红石题库：单选必答1题，多选必答1题
- 指令题库：单选必答1题，多选必答1题
- 服务器规则题库：单选必答3题，多选必答2题

## 答题规则

1. **分类选择**：
   - 用户需要选择自己擅长的题库分类
   - **最少选择1个分类，最多选择5个分类**
   - 可多选分类，系统将按比例随机抽取题目

2. **题目分配**：
   - 总共需要回答30道题：
     - 单选题：20道
     - 多选题：10道
     - 服务器规则题：5道（必选）

3. **必答题分配**：
   - **每个选择的分类都会随机分配1道单选必答题和1道多选必答题**
   - 服务器规则题也会随机分配1道单选必答题和1道多选必答题
   - 必答题是从该分类的所有必答题中随机抽取的

4. **通过率要求**：85%以上

5. **淘汰机制**：
   - 必答题答错直接淘汰
   - 答错后显示错误提示，需要重新开始答题

## 文件说明

- `题库总表.csv` - 包含所有题库的完整题目列表
- `建筑题库.csv` - 建筑分类的题目（50题）
- `生存题库.csv` - 生存分类的题目（50题）
- `养老题库.csv` - 养老分类的题目（60题）
- `红石题库.csv` - 红石分类的题目（60题）
- `指令题库.csv` - 指令分类的题目（60题）
- `服务器规则题库.csv` - 服务器规则题目（30题）

## 表格字段说明

| 字段名 | 说明 |
|--------|------|
| 题库分类 | 题目所属的分类 |
| 题目ID | 题目在分类中的编号 |
| 题目类型 | 单选题或多选题 |
| 是否必答 | 是/否，标记是否为必答题 |
| 题目内容 | 题目的具体内容 |
| 选项A | 第一个选项 |
| 选项B | 第二个选项 |
| 选项C | 第三个选项 |
| 选项D | 第四个选项 |
| 正确答案 | 正确答案（单选为字母，多选为字母组合） |
| 备注 | 题目的知识点说明 |

## 答案格式

- 单选题：A、B、C、D（单个字母）
- 多选题：ABCD（多个字母组合，如AB、ABC、ABCD等）

## 注意事项

1. 必答题会在答题界面显示"⚠️ 必答题"标记
2. 服务器规则题会显示"服务器规则"标记
3. 多选题会显示"多选题"标记
4. 答题时请仔细阅读题目，确保理解题意
5. 必答题答错会立即显示错误提示并结束答题
6. **选择分类时请注意：最少1个，最多5个**
