# Minecraft 服务器白名单管理系统完整文档

## 1. 项目概述

本项目是一个功能完善的 Minecraft 服务器白名单管理系统，专为 CloudTops 服务器设计。系统提供了完整的白名单申请、管理、监控功能，是一个专业级的服务器管理平台。

**核心功能：**
- 白名单申请与审核系统
- 智能答题测试系统
- 管理员后台管理
- 服务器状态监控
- 公告和活动管理
- 黑名单管理
- 投诉处理系统
- 数据统计分析

**技术特点：**
- 现代化前端技术栈
- 稳定的后端架构
- 完整的API系统
- 响应式设计
- 安全可靠的权限管理

## 3. 核心功能模块

### 3.1 白名单申请系统

**申请流程：**
1. **领域选择** - 玩家选择 1-5 个擅长的游戏领域
2. **规则测试** - 所有玩家必须完成服务器规则测试
3. **领域测试** - 根据选择的领域进行相应的测试
4. **信息填写** - 填写 Minecraft ID、QQ号、年龄等信息
5. **提交申请** - 管理员审核后加入白名单

**领域分类：**

| 领域 | 测试方式 | 测试内容 |
|------|----------|----------|

| 酿造 | 答题 | 药水配方、效果、材料 |
| PVP战斗 | 答题 | 战斗技巧、装备选择、战略 |
| 指令 | 答题 | 命令语法、参数、应用 |
| 探险 | 答题 | 地形识别、遗迹探索 |
| 下界 | 答题 | 下界探索、下界生物、下界资源 |
| 末地 | 答题 | 末地探索、末影龙战斗、末地资源 |
| 养老 | 答题 | 服务器规则、社区互动 |
