# 白名单系统测试清单

## 测试地址

- 主页：http://localhost:3000
- 申请页面：http://localhost:3000/apply
- 管理员后台：http://localhost:3000/admin
- 管理员审核：http://localhost:3000/admin/applications
- 管理员守则：http://localhost:3000/admin/guidelines

---

## 第一部分：用户申请流程

### 1. 选择题库
- [x] 打开申请页面
- [x] 选择"建筑"或"生电"题库（会有特殊提示）
- [x] 或者选择其他普通题库
- [x] 查看是否显示了解程度滑块
- [x] 查看是否有"返回主页"按钮

### 2. 开始答题提示弹窗
- [x] 点击"开始答题"后是否弹出大提示框
- [x] 提示框是否说明需要准备作品（选择了建筑/生电时）
- [x] 作品要求提示是否简洁（存档必选，照片视频二选一）
- [x] 点击"我已了解，开始答题"是否能正常进入答题

### 3. 答题过程
- [x] 题目是否正常显示
- [x] 题目类型是否显示为"主观题"（不是实景题）
- [x] 单选题是否能正常选择
- [x] 多选题是否能选择多个选项
- [x] 判断题是否正常
- [x] 能否切换上一题/下一题
- [x] 能否使用题目导航
- [x] 左下角是否有"返回重新选择"按钮

### 4. 返回重新选择
- [x] 点击"返回重新选择"按钮是否弹出确认框
- [x] 点击"取消"是否留在答题页面
- [x] 点击"确认返回"是否返回题库选择页面
- [x] 返回后已答题目是否被清空

### 5. 主观题测试
- [x] 主观题是否显示为文字输入框
- [x] 语音输入按钮是否可用（点击🎤按钮）
- [x] 点击语音输入按钮后，按钮是否变为红色闪烁+显示"录音中..."
- [x] 语音输入时，输入框placeholder是否显示"说话中..."
- [x] 对着麦克风说话，文字是否实时追加到输入框（不清空）
- [x] 再次点击按钮是否停止录音
- [x] 先输入一些文字，再语音输入，文字是否追加在后面
- [x] 说话停顿后再继续说，文字是否会重复（不应该重复）
- [x] 多段语音输入是否会正确追加（请听题 + 第一题 + 天马行空）
- [ ] 直接打字输入是否正常

### 6. 提交试卷
- [ ] 最后一题点击"提交试卷"后是否显示答题结果
- [ ] 是否显示得分和正确率
- [ ] 是否显示所有题目的答题详情
- [ ] 60分及以上是否显示"填写申请表"按钮
- [ ] 60分以下是否显示"重新答题"按钮

### 7. 填写申请表
- [ ] Minecraft ID输入是否正常
- [ ] QQ号输入是否正常
- [ ] 年龄、性别等下拉框是否正常
- [ ] 擅长类型是否根据选择的题库自动填充

### 8. 作品上传（选择了建筑/生电才显示）
- [ ] 是否显示三个上传区域：存档、照片、视频
- [ ] 存档：是否标记"* 必选"
- [ ] 照片：是否标记"* 至少3张"
- [ ] 视频：是否标记"可选"
- [ ] 能否选择照片并显示预览
- [ ] 能否删除已上传的照片
- [ ] 能否选择视频并显示预览
- [ ] 能否删除已上传的视频
- [ ] 能否选择存档并显示文件名
- [ ] 能否删除已上传的存档
- [ ] 不上传存档能否提交（应该不行）
- [ ] 上传存档但没有照片或视频能否提交（应该不行）

### 9. 提交申请
- [ ] 点击"提交申请"按钮
- [ ] 是否有加载提示
- [ ] 提交成功是否有提示
- [ ] 提交后是否跳转回主页

## 第二部分：管理员后台

### 1. 登录后台
- [ ] 打开 http://localhost:3000/admin
- [ ] 使用管理员账号登录
- [ ] 是否能看到左侧菜单

### 2. 白名单审核页面
- [ ] 能否看到待审核申请列表
- [ ] 能否切换到"我的审核记录"
- [ ] 点击"审核"按钮能否打开审核弹窗

### 3. 审核详情查看
- [ ] 能否看到玩家的游戏ID
- [ ] 能否看到玩家的联系方式（QQ号）
- [ ] 能否看到玩家年龄
- [ ] 能否看到答题得分
- [ ] 能否看到玩家选择的专业领域

### 4. 作品展示（重要！）
- [ ] 如果玩家选择了建筑/生电，是否显示"玩家作品"区域
- [ ] 能否看到玩家上传的作品图片
- [ ] 能否看到玩家上传的视频
- [ ] 能否看到玩家上传的存档
- [ ] 图片/视频显示是否正常

### 5. 主观题答案（重要！）
- [ ] 能否看到"其他信息"区域
- [ ] 能否看到主观题的答案
- [ ] 答案显示是否完整

### 6. 审核操作
- [ ] 能否填写审核备注
- [ ] 点击"通过"按钮能否正常通过
- [ ] 点击"拒绝"按钮能否正常拒绝（需要填写理由）
- [ ] 拒绝时是否会提示填写理由

### 7. 管理员守则
- [ ] 左侧菜单是否有"管理员守则"
- [ ] 点击能否打开守则页面
- [ ] 守则内容是否完整
